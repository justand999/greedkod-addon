# greedkod-addon
Расширение гридкода для браузера (скидки и промокоды)

Расширение для поиска скидок, купонов, промокодов и показа их прямо на сайте доступно на странице:
https://addons.mozilla.org/ru/firefox/addon/поиск-скидок-и-промокодов/

Скоро будет и для других браузеров, проверьте подвал сайта https://greedkod.ru на предмет ссылки "Расширение", возможно, оно уже доступно для вашего браузера.
